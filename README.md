📧 Email/SMS Spam Detection System

A Machine Learning-based web application that detects whether a message is Spam or Not Spam (Ham) using NLP techniques.

🚀 Features
📥 Input email/SMS message
🤖 Predicts spam or not spam using ML model
⚡ Fast real-time detection
📊 Simple and clean UI
🛠️ Tech Stack
Python
Flask
Machine Learning (NLP)
HTML, CSS
Pickle (Model serialization)

▶️ How to Run
git clone https://github.com/your-username/Spam-Classifier.git

cd Spam-Classifier

pip install -r requirements.txt

python app.py

Open in browser:

http://127.0.0.1:5000/
